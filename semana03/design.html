Entendendo o Grid e Contraste

O que é Contraste de Cores?

O contraste é a diferença de luminância ou cor que torna um objeto distinguível de outros. No design web, garantir um contraste adequado entre o texto e o fundo é vital para que todos os usuários consigam ler a informação sem esforço.

Dica de Acessibilidade

Sempre teste suas cores no DevTools ou em ferramentas online. O nível AA exige 4.5:1 para texto comum e AAA exige 7:1.

Lista de Verificação

  • Validar contraste das fontes
  • Usar unidades relativas (rem, em)
  • Estruturar com CSS Grid/Flexbox
  • Evitar elementos obsoletos
